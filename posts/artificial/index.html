Identify Target and Add to Hosts File

URL: http://10.10.11.74

Upon visiting the IP in a browser, we are redirected to:

http://artificial.htb

Add this domain to /etc/hosts:

sudo nano /etc/hosts
10.10.11.74 artificial.htb
